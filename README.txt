Functionality
-------------
-The game has three difficulty levels, which can be choosen from the main screen.
-The number of mines in the game vary based on the difficulty level chosen.
-The game consists of 9*9 grids of masked cells. 
-Based on the difficulty level, a number of cells will be randomly populated with a hidden mine.
-The cells can be opened by clicking it.
-Clicking/openning a cell with mine under it will make you lose the game.
-Clicking on a empty cell with a mine in the immediate neighbours, will open the cell and display a number indicating the number neighbouring cell that contains mine.
-Ckicking on a empty cell with no mine in the immediate neighbours, will cause a chain reacting of opening accross all directions surrounding the cell.
-The chain reaction opens cells till a border is reached or flag is reached or a cell containg neighbouring mine is reached.
-Suspected mines can be flagged/unflagged by long clicks. Flagged cells can't be openned making it safer to mark a suspected cell.
-The game is own when all the cells without mine is openned.
-On winning or losing the game, an alert pop up is displayed to inform user of the status. From the alert user can either choose to play the game one more time or go back to main screen.
-Through out the game user can either reset the game using the button or go back to main screen using back button.
-There is a timer on the top right corner to display or track the users gaming performance.
